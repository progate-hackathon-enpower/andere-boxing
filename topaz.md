# AI活用
・`.claude/CLAUDE.md` でプロジェクト方針を定義し、さらに `.claude/rules/` に Rust / TypeScript それぞれの言語別コーディング規約を配置。ファイルパスに応じて自動適用されるため、チーム全体のコードスタイルを統一した
・`claude-code-action` を使った PR 自動レビューワークフローを構築。バグ・セキュリティ・型安全性・規約準拠の観点でレビューし、`@claude` メンションで対話的な追加質問にも対応可能にした

## GitHub Agentic Workflow
2/13頃にローンチされたので導入してみた。
・PR マージ後に README の更新要否を AI が判断し、必要な場合のみ自動で更新 PR を作成する
・毎日定時に AI がリポジトリの状態を分析し「Repo Report」Issue を作成。`report` ラベルをトリガーに Discord へ自動配信する仕組みも組み合わせた。
